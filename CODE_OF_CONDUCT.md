# Code of conduct

We are committed to a welcoming, harassment-free project for everyone.
Participants should communicate respectfully, critique ideas rather than
people, assume good intent while addressing harmful impact, and avoid abusive,
discriminatory, or disruptive behavior.

Maintainers may remove content or restrict participation when behavior harms
the community. Conduct concerns should be reported privately to the repository
owner through GitHub until a dedicated moderation address is published.
